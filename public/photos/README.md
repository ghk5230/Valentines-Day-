# Photos

Place your photos here. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

Then update the filenames in `src/content/content.ts` â†’ `photos` and `timeline` arrays.

Example:
1. Copy `our-first-date.jpg` into this folder
2. In `content.ts`, change `{ filename: 'photo1.svg', caption: '...' }` to `{ filename: 'our-first-date.jpg', caption: 'Our first date' }`

Tip: For best quality on iPhone 17 Pro, use images at least 1200 px wide.
The placeholder `.svg` files can be deleted once you add your real photos.
